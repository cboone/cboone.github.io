A terminal UI for exploring 497 terminal progress indicators (378 spinners, 119 progress bars) from 26 open-source collections. Browse, preview animations in real time, search, and export entries as JSON.

## Installation

```bash
brew install cboone/tap/right-round
```

## Navigation

- `j` / `k` or arrow keys to move through the catalog
- `Tab` to switch between spinners and progress bars
- `/` to search
- `Enter` to copy the selected entry as JSON
- Side-by-side panels on wide terminals (100+ columns), sequential view on narrow ones

## Categories

Spinners are organized into twelve categories including braille, geometric, emoji, and text variants. Progress bars are grouped by visual style: blocks, decorative elements, and phased designs.
