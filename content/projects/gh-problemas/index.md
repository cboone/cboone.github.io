A terminal UI for triaging and managing GitHub issues. Uses your existing `gh` authentication.

## Installation

```bash
brew install cboone/tap/gh-problemas
```

## Usage

```bash
gh-problemas
```

Requires prior authentication with the GitHub CLI:

```bash
gh auth login
```
