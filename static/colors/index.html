Usage Guidelines

Backgrounds

Variable Use For
--code-bg List page backgrounds, inline code backgrounds
--code-block-bg Code block backgrounds, pre elements
--entry Cards, elevated surfaces, post entries (creates contrast on list pages)
--theme Single page backgrounds, base background color
--theme-rgb When you need --theme with opacity (e.g., rgba(var(--theme-rgb), 0.85))

Text

Variable Use For
--content Body text, main content
--gray h4/h5 headings, very muted text
--primary Headings, buttons, high-emphasis text
--secondary Metadata, descriptions, muted text, footer

Accents and Borders

Variable Use For
--border Element borders, dividers
--darkcolor Links, interactive elements, hover underlines, active states
--lightcolor Hover states, button backgrounds, secondary accents
--tertiary Borders, list markers, subtle accents, scroll-to-top button

Notice Components

Each notice type has three variables: --notice-{type}-bg, --notice-{type}-border, --notice-{type}-text

Type Use For
abstract Summaries, abstracts (green)
danger Danger, bug, failure warnings (red) - shared by .bug and .failure
example Examples (purple)
info Information callouts (blue)
note Notes (indigo)
question Questions (amber)
quote Quotations (gray)
success Success messages (green)
tip Tips (teal)
todo Todo items (fuchsia)
warning Warnings (orange)
